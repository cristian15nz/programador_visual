Dato

Son la representacion simbolica ya sea mediante numeros o letras de una recopilacion de informacion la cual puede ser cualitativa o cuantitativa. Se usan para proveer informacion pero por si solos no tienen sentido.

Informacion

Son un conjunto de datos entrelazados mediante conectores linguisticos para formar una idea o definicion

Base de datos

Es un conjunto de datos pertenecientes a un mismo contexto y almacenados sistematicamente para su posterior uso.

Servidor de Base de datos

Son los que se utilizan para los gestores de base de base de datos y que varios usuarios hagan varias operaciones sobre ellas al mismo tiempo, situados en lugares diferentes. Se puede acceder a ellas mediante terminales o equipos con un programa llamado cliente.

Gestor de Base de Datos

Son un conjunto de programas que manejan todo acceso a la base de datos, con el objetivo de servir de interfaz entre ésta, el usuario y las aplicaciones utilizadas.

  • Usuario normales
  • Programadores de aplicaciones
  • Usuarios sofisticados
  • Administradores de Base de Datos
Esquema de Base de Datos

Representa la configuracion logica de todo o parte de una base de dato relacional. Puede existir de dos formas: Como representacion visual y como un conjunto de formulas conocidas como restricciones de integridad que controlan una base de datos. Estas formulas se expresan en un lenguaje de definicion de datos, tal como SQL (Structured Query Language). Como parte de un diccionario de datos, un esquema indica como las entidades que comforman la base de datos se relacionan entre si, incluidas las tablas, las vistas, los procedimientos almacenados y muchos mas.
